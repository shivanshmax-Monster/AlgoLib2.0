About AlgoLib

AlgoLib is a library built to solve a common problem faced by competitive programmers: the repetitive task of writing standard algorithms from scratch.

Whether you are competing on LeetCode, CodeChef, or Codeforces, saving time on implementation can be the difference between a win and a loss. This website provides optimized, copy-paste-ready implementations of complex algorithms in Java, and C++. It serves as a quick reference tool for students and developers to visualize code logic and speed up their development workflow.

How to Contribute

AlgoLib is community-driven! If you have a more optimized version of an algorithm, or if you want to add a new algorithm that isn't listed here, follow these steps:

  • 1
    Fork the Repository: Go to our contribution repo on GitHub and fork the AlgoLib_Contribution repo.
  • 2
    Add Your Code: Add your algorithm file (e.g., xyzAlgo.java or xyzAlgo.cpp). Ensure code is clean.
  • 3
    Pull Request: Submit a Pull Request with a clear description of your changes.

*Alternatively, if you are not comfortable with Git, you can mail your code snippet directly to the Lead/Co Developer.

Rewards: Verified contributors will be officially listed in our "Contributors Hall of Fame" section on this website as a token of appreciation! View Contributors
